libscip2hat
============
北陽製レーザー測域センサ"URG"を利用するためのライブラリ


VERSION & UPDATE
-----------------
libscip2hat Release 0.1, April 08, 2015


SUPPORTED PLATFORMS
--------------------
-  OSX
-  Linux
-  Microsoft Windows
    with MinGW/Cygwin
         and pthreads_win32 (http://sources.redhat.com/pthreads-win32/)


BUILD
-----
 this project use cmake

 1.cd build
 2.cmake ..
 3.make
 4.sudo make install
